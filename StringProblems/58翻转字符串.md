## 题目一：翻转单词顺序。  
输入一个英文句子，翻转句子中单词的顺序，但是单词内字符的顺序不变。把标点字符和字符一样处理。  
思路:
1. 首先翻转句子中的所有字符，彻底逆序，包括单词内的字符也翻转。
2. 然后在翻转单词内的字符，保证单词内的字符是有序的。   
关键在于实现一个函数翻转字符中的一段。   
```
//翻转一段字符串
public static void reverse(char pBegin, char pEnd) {
    if(pBegin == null || pEnd == null) {
        return ;
    }
    
    while(pBegin < pEnd) {
        char temp = pBegin;
        pBegin = pEnd;
        pEnd = temp;
    }
}
```
