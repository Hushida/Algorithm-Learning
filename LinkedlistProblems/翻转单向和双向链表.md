# 翻转单向链表
```
public class Node {
   public int value;
   public Node next;
   public Node(int value) {
       this.value = value;
   }
}

public Node reverseList(Node head) {
    Node next = null;
    Node pre = null;
    while(head != null) {
        next = head.next; //保存下一个节点，防止链表断裂
        head.next = pre;
        pre = head; //将pre往后移动一位，head也后移一位
        head = next;
    }
    return pre;
}
```
